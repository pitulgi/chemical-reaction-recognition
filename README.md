# Классификация механизмов химических реакций с помощью нейронных сетей

## Описание проекта

Программа генерирует синтетические данные кинетики двух типов химических реакций и обучает нейронные сети классифицировать механизм реакции по временным рядам концентраций.

### Типы реакций

| **Тип 1** | Последовательная: A → B → C → D |
| **Тип 2** | Гибридная: A → B → D и A → B → C → D (параллельно-последовательная) |

### Кинетические константы

**Тип 1 (последовательная):**
- k₁ = 2.0 (A → B)
- k₂ = 1.5 (B → C)
- k₃ = 1.0 (C → D)

**Тип 2 (гибридная):**
- k₁ = 1.8 (A → B)
- k₂ = 0.7 (B → D)
- k₃ = 0.5 (B → C)
- k₄ = 0.9 (C → D)

## 2. Запуск
###Запустите первый скрипт для создания датасета: 
```bash
python generate_data.py
```
### Результат
Сгенерируется 4000 образцов (по 2000 на каждый тип реакции)

Для каждого образца начальная концентрация A₀ выбирается случайно от 10 до 100 моль/л

Концентрации B, C, D в начальный момент равны 0

Решается система ОДУ методом Эйлера с шагом 0.1 с (всего 20 временных точек)

Добавляется шум ±5% для имитации экспериментальных погрешностей

Результат сохраняется в data.csv и data.xlsx

###Запустите второй скрипт:
```bash
python main.py
```
### Результат
Загрузка данных из data.csv

Нормализация признаков

Разделение на обучающую и тестовую выборки 80/20

Обучение двух нейронных сетей:

Однослойный перцептрон (без скрытых слоев, логистическая активация)

Многослойный перцептрон (2 скрытых слоя: 64 и 32 нейрона)

Вывод точности классификации

Визуализация примеров траекторий

